# Exercícios de Lógica de Programação

Este repositório contém exercícios de lógica de programação baseados nos conteúdos dos prints fornecidos.

Os algoritmos foram desenvolvidos utilizando:

*  **VisuAlg** (pseudocódigo)
*  **Flowgorithm** (fluxogramas)

---

## Objetivo

Praticar conceitos fundamentais de programação, como:

* Entrada e saída de dados
* Operações matemáticas
* Estruturas condicionais
* Tomada de decisão

---

## Tecnologias utilizadas

* **VisuAlg** → para desenvolvimento dos algoritmos em pseudocódigo
* **Flowgorithm** → para representação visual em fluxogramas
